# Get in contact

NOMAD, `pynxtools`, and `pynxtools-em` are open source projects that warmly welcome community projects, contributions, suggestions, bug fixes, and constructive feedback. `pynxtools` is build mainly within FAIRmat Area B - Experiment.

You can reach us by different channels. You can send an email directly to one of the main contributors:

| Name | E-mail     | Github profiles |
|------|------------|-----------------|
| The NOMAD Authors | [fairmat@physik.hu-berlin.de](mailto:fairmat@physik.hu-berlin.de) |  |

Alternatively, you can also:
- Open an issue on [GitHub](https://github.com/FAIRmat-NFDI/pynxtools-em/issues)
- Join the [NOMAD discord channel](https://discord.gg/Gyzx3ukUw8) and ask us there directly.
